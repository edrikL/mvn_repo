# mvn repo

#### 介绍
个人 mvn repo


## Usage

add below to pom.xml


```xml

    <repositories>
        <repository>
            <id>edrik-mvnrepo</id>
            <name>edrik-mvnrepo</name>
            <url>https://raw.githubusercontent.com/edrikL/mvn_repo/master/repository</url>
        </repository>

    </repositories>
```
